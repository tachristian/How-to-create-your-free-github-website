How to Create Your Own Free GitHub Website

In addition to being a great tool for versioning files, Git is a powerful collaboration platform for sharing and working on documents as part of a team. Git, and backup tools such as GitHub, solve the common problems that are encountered when collaborating on a document.

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere. Over 94 million developers shape the future of software, together.

Create Your Site

  1. Open a GitHub account.
  2. Click Repositories.
  3. Click New.
  4. Name your repository.
  5. Click Create repository.
  6. Under Code > Quick Setup > Click Get started by creating a new file.
  7. Name your file: index.html
  8. Add any text you want to show on your site.
  9. Scroll down and click Commit new file.

Publish Your Site

  1. Click Settings.
  2. Under General > Code and Automation > Click Pages.
  3. Under Branch, click the dropdown and select main.
  4. Click Save.
