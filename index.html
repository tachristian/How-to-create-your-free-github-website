How to Create Your Own Free GitHub Website

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere. Over 94 million developers shape the future of software, together.

##Create Your Site

Open a GitHub account. Click Repositories. Click New. Name your repository. Click Create repository. Under Code > Quick Setup > Get started by creating a new file. Name your file: index.html Add any text you want to show on your site. Scroll down and click Commit new file.

##

Publish Your Site
